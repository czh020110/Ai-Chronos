# 本地开发环境

## 必需工具

- Node.js：>=18；检查命令：`node --version`
- npm：随 Node.js 安装；检查命令：`npm --version`

## 初始化步骤

```bash
cd frontend && npm install && npm run dev
```

## 环境变量

当前暂无。

## 外部服务

当前暂无（第一阶段为纯前端静态展示，使用内置示例数据 `frontend/lib/data.ts`）。

## 已知环境坑

- 开发服务器端口自动递增：`next dev` 默认使用 3000 端口，被占用时会自动尝试 3001、3002 等。启动后请以终端输出的实际端口为准。
- 移动端触摸拖拽未适配：当前拖拽滚动仅支持鼠标事件（mousedown/mousemove/mouseup），移动端触摸时间线节点交互需后续适配。

## 技术栈

- 框架：Next.js 14.2（App Router）
- 渲染：React 18.3
- 样式：TailwindCSS 3.4 + CSS 自定义属性
- 动画：Framer Motion 11 + Canvas 2D
- Markdown 渲染：react-markdown 9 + remark-gfm
- 日期处理：date-fns 3.6
- 语言：TypeScript 5.4

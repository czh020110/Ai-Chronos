# 常用命令

## 安装依赖

```bash
cd frontend && npm install
```

## 启动开发服务

```bash
cd frontend && npm run dev
```

开发服务会自动寻找可用端口（默认 3000，被占用时递增）。启动后终端会显示实际端口。

## 类型检查

```bash
cd frontend && npx tsc --noEmit
```

## Lint / 格式化

```bash
cd frontend && npm run lint
```

## 构建

```bash
cd frontend && npm run build
```

# 核心数据流

## 前端钻取式时间线交互链路

### 本次更新原因

- 关联 Commit：前端 UI 视觉重构（星空→认知地层/神经星图）
- 更新原因：前端从普通星空时间线升级为"认知地层/神经星图"式界面，重做了动态背景系统、新增左右侧叙事面板与坐标信息面板、重构时间线视觉，需同步数据流描述。

### 链路目标

用户在年/月/日三层视图间钻取浏览 AI 事件时间线，点击节点查看 Markdown 详情，通过搜索面板快速定位事件。

### 修改前数据流

新增链路，修改前无。

### 修改后数据流

[页面入口 `app/page.tsx`] → [状态管理 (drill, selectedEvent, searchOpen)] → [组件渲染]
  → [StarField: 动态背景渲染（星云/粒子/流动光带/扫描光效）]
  → [左侧叙事面板: 项目标题、事件统计摘要（时间跨度/事件数/高影响力数/领域数）]
  → [右侧坐标面板: 当前钻取层级描述、锚点事件标题与影响力进度条]
  → [NavBar: 触发搜索打开]
  → [Timeline: 根据 drill 渲染年/月/日节点（纪年星图/月度透镜/日级轨迹）] → [节点被点击] → 分支：
    ├─ 年份/月份节点 → [onDrillDown] → [更新 drill 状态] → [Timeline 重新渲染下一层级]
    └─ 日节点/事件球 → [onNodeClick] → [更新 selectedEvent + drawerOpen] → [DetailDrawer 显示 Markdown 详情]
  → [SearchPanel: 打开搜索面板 → 输入关键词 → 选择命中事件] → [onSearchSelect] → [打开 DetailDrawer + 高亮对应节点]
  → [DrillControls: 底部钻取面包屑 + ViewToggle: 视图切换]

### 输入

- `AIEvent[]` 事件数组
  - 来源：`frontend/lib/data.ts` 中的 `sampleEvents`（静态示例数据）
  - 关键字段：id, title, event_date, content_md, tags, source_urls, impact_score, category, status
  - 约束：第一阶段无后端，事件数据在组件初始化时一次性加载

### 处理步骤

1. `frontend/app/page.tsx`::`Home`
   - 职责：顶层状态容器，管理 drill、selectedEvent、drawerOpen、searchOpen、viewMode、highlightedId；同时通过 `useMemo` 计算左侧叙事面板统计指标（span、eventCount、highImpact、categoryCount、topEvent）和右侧坐标面板当前钻取描述。
   - 调用方：Next.js App Router（`layout.tsx` → `page.tsx`）
   - 被调用方：StarField, NavBar, 左侧叙事面板（内联 JSX）, 右侧坐标面板（内联 JSX）, Timeline, DetailDrawer, SearchPanel, DrillControls, ViewToggle
   - 调用时机：页面首次渲染
   - 输入：无外部输入，使用 `useState<AIEvent[]>(sampleEvents)` 初始化
   - 输出：通过 props 向子组件分发状态与回调；左侧面板渲染项目标题、统计指标卡片（时间跨度/事件数/高影响力数/领域数）；右侧面板渲染坐标层级标签、当前钻取描述与锚点事件影响力进度条
   - 状态变化：drill 状态通过 `handleDrillDown` / `handleDrillUp` 切换层级；searchOpen 通过 `Ctrl/Cmd+K` 快捷键或 NavBar 按钮切换
   - 异常分支：无

2. `frontend/components/timeline/Timeline.tsx`::`Timeline`
   - 职责：根据 `drill.level` 计算布局节点（年份间距 160px，月份按 `winW/14`，日期按 `winW/(daysInMonth+4)`），渲染"纪年星图 / 月度透镜 / 日级轨迹"三层视觉风格；TimelineCanvas 组件绘制多层级辉光轨道、节点光晕与连线；DOM 节点带 Spring 入场动效和信息卡。
   - 调用方：`app/page.tsx::Home`
   - 被调用方：TimelineCanvas（Canvas 2D 轨道背景）、GridView、ListView
   - 调用时机：页面渲染、drill 层级切换、viewMode 切换
   - 输入：events（完整事件列表）、drill（当前钻取状态）、viewMode（timeline/grid/list）、onNodeClick、onDrillDown、highlightedId
   - 输出：Framer Motion 动画节点（Spring 入场动画）、Canvas 背景（轨道辉光、节点光晕、多层级波纹、当前时刻标记）
   - 状态变化：内部维护 `scrollX`（RAF 惯性滚动）、`hoveredNode`（Tooltip 显示）、`transitioning`（层级切换标记）
   - 异常分支：viewMode 非 "timeline" 时直接渲染 GridView 或 ListView

3. `frontend/components/ui/DrillControls.tsx`::`DrillControls`
   - 职责：底部面包屑导航，显示当前层级路径（全部年份 / 2024年 / 1月），提供"返回上级"按钮
   - 调用方：`app/page.tsx::Home`
   - 被调用方：无
   - 调用时机：drill 状态不为 "year" 时渲染返回按钮
   - 输入：drill, onDrillUp
   - 输出：层级指示点（三点分别对应年/月/日，当前层金色发光）
   - 异常分支：无

4. `frontend/components/search/SearchPanel.tsx`::`SearchPanel`
   - 职责：搜索面板，支持关键词匹配（标题/标签/内容），键盘导航（上下箭头 + Enter）
   - 调用方：`app/page.tsx::Home`
   - 被调用方：无
   - 调用时机：isOpen 为 true 时渲染
   - 输入：events, isOpen, onClose, onSelect
   - 输出：最多 8 条搜索结果（含标题、日期、标签、影响力分数）；选中后调用 onSelect
   - 状态变化：内部维护 query 和 selectedIdx
   - 异常分支：无匹配时显示"未找到相关事件"

5. `frontend/components/drawer/DetailDrawer.tsx`::`DetailDrawer`
   - 职责：右侧滑入式 Markdown 详情抽屉，渲染事件摘要、标签、影响力条、参考来源
   - 调用方：`app/page.tsx::Home`
   - 被调用方：react-markdown + remark-gfm
   - 调用时机：drawerOpen 为 true 且有 selectedEvent 时渲染
   - 输入：event, isOpen, onClose（Esc 键关闭）
   - 输出：Markdown 渲染内容、影响力进度条动画、来源链接列表
   - 异常分支：event 为 null 时不渲染

6. `frontend/components/background/StarField.tsx`::`StarField`
   - 职责：复合动态背景系统（Canvas 2D，RAF 循环），包含：
     - 多层径向渐变底图（深空蓝紫渐变）
     - 流动极光带（蓝/金/紫三色正弦波光带，screen 混合模式）
     - 神经网络粒子系统（80–140 个三色调粒子，按 depth 分速度漂移，带脉冲闪烁和光晕）
     - 粒子间连接丝（近距离高深度粒子间半透明连线，模拟神经网络）
     - 扫描光效（从左至右的金色渐隐光柱）
     - 暗角遮罩（径向渐变压暗边缘）
   - 调用方：`app/page.tsx::Home`
   - 被调用方：无
   - 调用时机：页面渲染即启动，resize 时销毁重建（160ms 防抖），支持 HiDPI（devicePixelRatio 适配）
   - 异常分支：无

7. `frontend/app/page.tsx`::左侧叙事面板（内联 JSX）
   - 职责：固定定位左侧信息面板（xl 断点以上显示），展示 "Synthetic Memory Atlas" 副标题、项目主标题 "AI Chronos / 智纪元"、说明文字；通过 `useMemo` 统计 span（时间范围）、eventCount（事件总数）、highImpact（影响力 >=95 的事件数）、categoryCount（领域数）四项指标，以玻璃卡片网格展示。
   - 调用方：`app/page.tsx::Home`（内联）
   - 被调用方：无
   - 调用时机：页面首次渲染（有数据后秒出）
   - 输入：由 `stats` 派生
   - 输出：Framer Motion 淡入滑入动效的玻璃面板，含项目标识与统计卡片
   - 异常分支：无

8. `frontend/app/page.tsx`::右侧坐标面板（内联 JSX）
   - 职责：固定定位右侧信息面板（2xl 断点以上显示），实时反映当前钻取层级标签（Year Atlas / Month Lens / Day Trace）、当前坐标描述、锚点事件标题与影响力进度条（动画填充）。
   - 调用方：`app/page.tsx::Home`（内联）
   - 被调用方：无
   - 调用时机：drill 状态变化时实时更新坐标描述
   - 输入：drill.level, activeCoordinate (派生), stats.topEvent
   - 输出：Framer Motion 淡入滑入动效的玻璃面板，含层级标签、坐标文字与影响力进度条动画
   - 异常分支：无

### 状态流转

- 钻取路径：`year` 级（全部年份）→ 点击年份 → `month` 级（该年 12 个月）→ 点击月份 → `day` 级（该月每日视图）→ 点击日节点 → 打开 DetailDrawer
- 返回路径：点击 DrillControls "返回上级" 逐级退回 year 级
- 搜索路径：打开搜索面板 → 输入关键词 → 选择结果 → 打开 DetailDrawer 并 3 秒高亮对应节点 → 高亮自动消退

### 输出

- 用户可见：左侧叙事面板（项目标识 + 统计卡片）、右侧坐标面板（层级标签 + 坐标 + 锚点事件）、年/月/日钻取时间线（Canvas 轨道背景 + DOM 节点 + Spring 入场动效）、Markdown 详情抽屉、网格/列表视图、搜索面板
- 用户不可见：无后端写入、无数据持久化（当前阶段）

### 错误分支

- 搜索无结果：显示"未找到相关事件"提示，不影响页面其他交互
- 空月份/空日期节点：渲染为半透明虚节点，不可点击，标签显示无事件
